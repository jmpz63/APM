# Manufacturing Cell Specs (Hybrid Rotatable Table)

This component houses the specifications for the stationary, rotatable worktable hybrid cell:
- Robot placement of lumber/sheathing
- XY gantry nailing/stapling
- 0°/180° rotation with positive mechanical locks

Files
- `Workcell_Spec.md` — cell architecture, safety, controls, IO, cycle targets
- `Rotatable_Table_Spec.md` — mechanical design for the rotatable table
- `panel_recipe.schema.json` — JSON schema for panel geometry and nail patterns

See project hub: `PROJECTS/Active/WallPanelization/README.md`.
